\tituloSeccion{Objetivos}
\begin{itemize}
    \item Comprobar experimentalmente la relación entre tensión, corriente y resistencia.
    \item Aplicar el formato del informe para presentar resultados de laboratorio.
\end{itemize}

\tituloSeccion{Marco Teórico}
La Ley de Ohm establece que $V = IR$, donde $V$ es el voltaje, $I$ la corriente y $R$ la resistencia.

\tituloSeccion{Materiales}
\begin{itemize}
    \item Fuente de voltaje DC.
    \item Resistencias de prueba.
    \item Multímetro digital.
    \item Protoboard y cables de conexión.
\end{itemize}

\tituloSeccion{Procedimiento}
\begin{enumerate}
    \item Se armó el circuito resistivo sobre la protoboard.
    \item Se midió la corriente para distintos valores de voltaje.
    \item Se compararon los resultados con el valor teórico esperado.
\end{enumerate}
